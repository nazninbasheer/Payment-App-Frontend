This is the frontend of the Payment Collection App built using React Native with Expo.  
It allows users to view loan details and submit EMI payments.

## Tech Stack
- React Native
- Expo
- JavaScript

## Features
- Display loan details
- EMI payment form
- Payment confirmation message

## Run the App

Install dependencies:
npm install

Start Expo:
npx expo start

Run on:
- Android Emulator
- Physical device using Expo Go
- Web browser (optional)

## Build APK
eas build -p android --profile apk

## Notes
- Frontend communicates with backend APIs
- Backend must be accessible for full functionality
